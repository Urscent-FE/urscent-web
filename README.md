# Urscent web

머신러닝 알고리즘을 활용해 개인화된 향수 추천을 제공하는 웹

- [팀원 소개](#팀원-소개)
- [기술 스택](#기술-스택)

### 구현 기능 사항

- 검색 기능 : 향수명 또는 향수 정보를 통해 원하는 향수 검색
- 추천 기능 : 성별, 좋아요, 취향을 분석하여 추천을 제공
- 향수 상세 페이지 : 향수의 상세한 정보 제공
- 커뮤니티 기능 : 댓글로 의견을 남기기 제공
